Users Manual
============

.. _user-team:

Team
----

- Insert plugin “basketball-bund.net – Team”

- Tab **Options**:

  - Select the **Team**

  - Select what to **Display** (table, results, playing schedule, statistics)

  - Save and refresh

  - Select the **Columns** to be displayed

  - For results and playing schedule: **Select content** to display. ("Other teams by checkbox" adds a checkbox to the top of the table. By default only the own team will be displayed, by checking the box all teams will be displayed.)

- Tab **Layout**:

  - **Stress team** (stressed team is written in bold letters)

    - **Definded by TypoScript**: (default) use the team definde in TypoScript setup
    - **Yes**: save and refresh and enter the team to be stressed in filed **Stress team:** below
    - **No**: no team will be stressed

  - **Border** (display a table-border)

    - **Definded by TypoScript**: (default) use the setting in TypoScript setup
    - **Yes**: display border
    - **No**: display no border

  - **Allow linebreak in table-rows**

    - **Definded by TypoScript**: (default) use the setting in TypoScript setup
    - **Yes**: allow linebreaks (breaks the output in some cases)
    - **No**: do not allow linebreaks
